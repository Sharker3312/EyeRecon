# EyeRecon

Subdomain search tool

## Starting 🚀

```
git clone https://github.com/Sharker3312/EyeRecon.git
cd EyeRecon
./eyerecon.sh 
```

### Prerequisites 📋

You need to have these tools installed
* assetfinder 
* amass 
* waybackurls 
* findomain 
* sublist3r 
* gauplus 
* httpx 
* gowitness
